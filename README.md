# Birthday Surprise Website

An emotional, cinematic, full-screen friendship birthday experience built with React, Tailwind, Framer Motion, and GSAP.

## Run

1. Install dependencies: npm install
2. Start dev server: npm run dev
3. Build production: npm run build

## Personalization Placeholders

Update placeholders inside src/App.jsx with your real media:

1. Photos: /placeholders/memory-1.jpg, /placeholders/memory-2.jpg, /placeholders/memory-3.jpg
2. Background music: /placeholders/birthday-song.mp3
3. Memory video: /placeholders/our-memories.mp4
4. Video poster: /placeholders/video-poster.jpg
5. Letter text: replace the [PLACEHOLDER: Write your personal birthday letter here.] paragraph
6. Spotify track card text
7. Voice note card text/link
8. Scrapbook card content

## Notes

1. The page includes heavy animations, glassmorphism cards, floating particles, custom cursor glow, confetti clicks, and section storytelling.
2. All sections are mobile responsive and designed for smooth scrolling.
